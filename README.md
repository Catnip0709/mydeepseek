# 免费、极简、私密的 DeepSeek 网页客户端

## ✨ 三大核心理念

- **免费无广，零中间费用**： 所有充值仅在 [DeepSeek 官方平台](https://platform.deepseek.com/) 完成，本站不收取任何费用，没有任何广告
- **无需注册，无需登录**：打开浏览器即可使用，输入 API Key 马上开始对话
- **数据本地存储，隐私安全**：无个人服务器，不收集、不上传、不存储任何用户信息与聊天记录，所有数据仅保存在你的浏览器本地

> 所有对话请求直接发往 DeepSeek 官方 API 接口，本站不经过、不拦截、不记录任何通信内容。


## 🚀 快速开始

1. 前往 [DeepSeek 开放平台](https://platform.deepseek.com/) 注册并创建 API Key，在官方平台完成充值
2. 用浏览器打开本站：**[catnip0709.github.io/mydeepseek](https://catnip0709.github.io/mydeepseek/)**
3. 输入你的 API Key，即可开始对话


## 🎯 功能一览

### 💬 智能对话
- 支持 **DeepSeek-V3**（通用对话）和 **DeepSeek-R1**（深度思考）双模型切换
- 输入框上方「深度思考 R1」快捷按钮，一键切换模型，无需进入设置
- 流式输出，实时显示 AI 回复过程，群聊同样支持流式显示
- R1 模型支持展示完整思考链（Reasoning Content）
- 一键编辑已发送消息并重新生成，所有历史版本均可浏览切换（❮ / ❯）
- 支持回复特定消息，精准上下文关联（群聊中可回复指定角色）

### 🎭 角色扮演
- **角色卡系统**：自定义角色名字、性格、背景、外貌、说话风格、口头禅等
- **AI 智能生成角色卡**：输入一句话简要描述，AI 自动生成完整角色设定
- **单角色对话**：与指定角色进行沉浸式一对一对话
- **背景信息设定**：自定义你的角色名称和对话场景背景，AI 始终记住（所有会话类型通用）

### 👥 群聊模式
- 创建多角色群聊，AI 智能路由判断哪些角色应该回应
- 角色之间支持多轮追问互动，模拟真实群聊氛围
- 支持回复特定角色，触发精准回应，输入框显示回复引用条
- 场景感知：角色会根据你描述的在场/离场状态决定是否发言

### 📝 指令系统
- **指令市场**：浏览随机指令，一键插入新对话；空对话时自动引导前往指令市场
- **指令管理**：保存、编辑、复制常用指令
- **AI 智能优化**：对已有指令进行 AI 辅助润色，对比预览后决定是否替换
- **AI 智能生成**：输入关键词或描述，AI 自动生成角色扮演指令

### 🧠 智能记忆
- **滑动窗口摘要**：对话超过 40 条时自动压缩早期对话为摘要，保留最近 20 条原文，节省 Token
- **全量发送模式**：可选发送全部历史消息，不做摘要
- 摘要自动滚动更新，支持版本化迁移，无需手动管理

### 📎 文件上传
- **TXT 文件上传**：上传文本文件作为对话上下文
- 智能发送模式：5000 字以内全文发送；5000-25000 字自动调用 R1 生成摘要后发送；超过 25000 字提示超出上限

### 🔍 搜索与导出
- **对话内容搜索**：支持 Ctrl+F 快捷键，高亮匹配结果并逐条跳转
- **对话导出**：支持导出全部对话或仅 AI 回复，可选是否包含思考过程
- 每个 Tab 标签自带快捷导出按钮，无需切换会话即可导出
- 导出为 TXT 文件，方便保存和分享

### ⚙️ 个性化设置
- **日间/夜间模式**：一键切换明暗主题
- **字号调节**：5 档字号自由选择
- **Token 预估**：实时显示输入和历史消息的 Token 用量
- **Token 上限警告**：接近 128K 上限时自动提醒，并提供一键复制总结指令
- **存储用量监控**：显示本地存储使用情况，接近上限时自动提醒

### 🛡️ 稳定可靠
- **生成状态标记**：AI 回复显示完成 / 中断 / 超时三种状态
- **后台自动中断**：浏览器切到后台时自动中断生成，返回前台时 Toast 提示可重新生成
- **请求超时保护**：120 秒无数据自动中断，避免无限等待
- **数据自动修复**：异常数据启动时自动修复，极端情况可一键重置

### 📱 移动端适配
- 支持触摸手势（左滑打开侧边栏）
- 安全区域适配（刘海屏、底部横条）
- 响应式布局，手机、平板、桌面均可使用

## 🛠️ 技术栈

- **纯前端单页应用**，无需后端服务
- 原生 JavaScript（ES Modules）
- [Tailwind CSS](https://tailwindcss.com/) 样式框架
- [Marked.js](https://marked.js.org/) + [DOMPurify](https://github.com/cure53/DOMPurify) Markdown 渲染
- 数据持久化：浏览器 localStorage
- 流式通信：Fetch API + ReadableStream
