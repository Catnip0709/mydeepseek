# mydeepseek
极简、私密、无第三方的 DeepSeek 网页客户端

## 使用指南
1. 登录 DeepSeek 官网：https://platform.deepseek.com/，创建 API Key，并在官方平台完成充值。
2. 用浏览器打开：https://catnip0709.github.io/mydeepseek/，输入你的 API Key 即可开始对话。

## 亮点
1. 无需注册、无需登录，打开即用
2. 完全免费无广告，**不收取任何中间费用**，所有充值仅在 DeepSeek 官方完成
3. 无个人服务器，**不收集、不上传、不存储任何用户信息与聊天记录**，真正隐私安全
